# Premium UI Analyzer Skill

## Purpose
Analyze a reference website and apply its UI/UX direction to the current project without copying it directly.

The goal is inspiration and design direction only.

---

# What To Analyze

When given a website URL:

Analyze:
- Layout structure
- Visual hierarchy
- Typography feel
- Color palette
- Shadows and glow usage
- Card design
- Border radius
- Button styles
- Hover interactions
- Animations and motion
- Scroll behavior
- Section spacing
- Premium feeling
- Mobile responsiveness
- Navigation style
- Hero section composition
- Content density

---

# Important Rules

DO NOT:
- Copy branding
- Copy text
- Copy assets
- Clone the website exactly

ONLY:
- Apply the same quality level
- Apply similar UI direction
- Improve the existing website

---

# Project Context

This project is:
Cre8ors Hub

Stack:
- React + Vite
- React Router
- Custom CSS
- Vercel deployment

Style:
- Dark luxury
- Black / charcoal backgrounds

- Cinematic premium UI

---

# Workflow

1. Analyze the reference website.
2. Create a short UI analysis summary.
3. Suggest improvements for Cre8ors Hub.
4. Apply targeted changes only.
5. Keep the current routing and structure intact.
6. Maintain performance.
7. Verify responsiveness.

---

# Technical Rules

- Do not rewrite the whole app.
- Reuse existing components.
- Prefer modular CSS.
- Keep animations lightweight.
- Avoid bulky UI.
- Keep spacing clean and elegant.

---
